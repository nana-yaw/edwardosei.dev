# Portfolio Polish & SEO — Design Doc

**Date:** 2026-02-27
**Scope:** 7 improvements to the edwardosei.dev portfolio — SEO foundation, OG metadata, JSON-LD, favicon fix, reduced-motion audit, custom 404, and mobile responsiveness audit.

---

## 1. SEO: sitemap.ts + robots.ts

**Files:** `src/app/sitemap.ts`, `src/app/robots.ts`

`sitemap.ts` returns a single entry for `/portfolio` (the only real page — `/` redirects). Uses Next.js App Router convention for static generation.

`robots.ts` allows all crawlers and points to `https://edwardosei.dev/sitemap.xml`.

## 2. OG Metadata Fix + Portfolio-Specific Metadata

**Files:** `src/app/layout.tsx`, `src/app/portfolio/page.tsx`

**Problem:** layout.tsx manually sets `openGraph.images` to `/og-image.png` which doesn't exist as a static file. The actual OG image is generated by `src/app/opengraph-image.tsx` (dynamic route).

**Fix:** Remove the manual `openGraph.images` array from layout metadata — Next.js auto-discovers the co-located `opengraph-image.tsx` route. Add a `metadata` export to `portfolio/page.tsx` with portfolio-specific title and description.

## 3. JSON-LD Person Schema

**File:** `src/app/portfolio/page.tsx`

Add a `<script type="application/ld+json">` with `Person` schema in the portfolio page. Data sourced from `profile.ts`:
- name, jobTitle, worksFor (RentPost), url, sameAs (GitHub, LinkedIn)
- alumniOf (University of Cape Coast)

Server component renders it — no client JS cost.

## 4. Favicon: Generate apple-touch-icon.png

**File:** `public/apple-touch-icon.png`

layout.tsx references `apple-touch-icon.png` in its icons metadata but the file doesn't exist (404 on iOS). Generate a 180×180 PNG with the "EO" initials on a dark background matching the devONE brand.

Approach: Create via canvas script or inline SVG-to-PNG conversion. Simple enough to hand-craft as an SVG and convert.

## 5. prefers-reduced-motion Audit + Fix

**Files:** Multiple theme components using Framer Motion

StoryOrchestrator already has `@media (prefers-reduced-motion: reduce)` CSS that disables carousel transitions. But Framer Motion animations (fade-in, slide-up reveals across all 4 themes) are not gated.

**Fix:** Add Framer Motion's `useReducedMotion()` hook or set `MotionConfig reducedMotion="user"` at the layout level. This makes all `motion.*` components respect the OS preference automatically with zero per-component changes.

## 6. Custom 404 Page

**File:** `src/app/not-found.tsx`

Theme-adaptive 404 using CSS custom properties (same pattern as AlsoBuiltCard). Shows:
- "404" large heading
- "Page not found" subtext
- "Back to portfolio" link → `/portfolio`

~30 lines, no animations, respects current theme.
