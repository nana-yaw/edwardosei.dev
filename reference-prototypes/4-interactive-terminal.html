# Architecture Overview

Client Layer
Next.js 16 App Router (Server + Client Components)
React 19 with Suspense boundaries
Tailwind CSS 4 + Radix UI primitives

Real-time Layer
Convex reactive queries (auto-updating subscriptions)
Optimistic mutations with conflict resolution
Cron scheduler for automated service lifecycle

Auth & Security Layer
Magic link auth via Resend
5-tier RBAC: super_admin → pastor → guide → staff → viewer
ABAC with scope/time/IP conditions
PII projection filtering by role

Data Layer
30+ tables with denormalized counters
Soft delete with registry tracking
Ghana phone normalization (+233)
Integrity chain audit log (SHA-256)
